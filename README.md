# Tic-Tac-Toe Web Application

A responsive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The application allows two players to play the classic game with automatic win detection, draw detection, and game reset functionality.

## Features

- Two-Player Gameplay
- Interactive 3×3 Game Board
- Automatic Win Detection
- Draw Detection
- Reset/New Game Functionality
- Responsive Design
- Clean & Interactive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Objective

To build an interactive Tic-Tac-Toe game that manages player turns, tracks the game state, detects winners or draws, and provides a smooth user experience.
